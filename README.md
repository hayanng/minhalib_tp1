# minhalib_tp1
Trabalho acadêmico de funções em C++

No final da resolução das listas de exercícios foi feito esse trabalho juntando todas as funções feitas nos exercícios, abordando passagem de parâmetros por referência, por ponteiro e tipos de funções.

1- O trabalho inicializa com uma tela de apresentação que pede para pressionar o "ESPAÇO" para continuar.

2- Então é aberto uma tela com opções para testar as funções feitas

3- Ao selecionar a opção referente a função é direcionado para fazer o teste da mesma.

4- Cada opção tem a possibilidade de testar quantas vezes quiser

5- Ao pressionar 0 o programa voltará ao quadro de opções

6- Ao pressionar 0 novamente irá sair do programa apresentando uma mensagem de despedida
